# MagentoAssignment-2

## Description
This Magento 2 Wallet Module allows customers to add, transfer, and use wallet funds for purchases. It includes OTP verification, transaction history, and an admin panel for managing wallet balances.
## Technical Requirements
1. PHP
2. MY SQL
3. Nginx
4. Magento 2
5. Html

## Getting Started

1. Clone the project

```bash
git clone https://github.com/Altaf192816/Magento-Assignment-2.git

```
2. Install PHP,MySql and Nginx
> [!TIP]
> For detail installation process visit this [link](https://www.digitalocean.com/community/tutorials/how-to-install-lamp-stack-on-ubuntu)
3. Install Magento
